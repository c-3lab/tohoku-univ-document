## １ GoogleColaboratoryログイン
1. [GoogleColaboratory](https://colab.research.google.com/)のページへ移動

2. 右上にある「ログイン」をクリック

3. 対象のアカウントを選択

4. パスワードを入力して「次へ」をクリック

## ２ ノートブックの作成
1. 「ノートブックを新規作成」をクリック

2. ノートブックが開くのでセル内に `1+1` を記載して実行ボタンをクリック

3. 2 という結果が出力されていることを確認する
